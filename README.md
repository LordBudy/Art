## 🚀 Основные функции
- 🔍 **Поиск** картин и фотографий по ключевым словам  
- ❤️ **Добавление в избранное** для быстрого доступа  
- 🖼️ **Просмотр в полном размере**  
- 🪄 **Установка на обои** одним касанием  
- ☁️ Источники изображений — **The Met Museum API**

---

## 🧠 Архитектура
Приложение построено на современных технологиях Android:

- 🧩 **Kotlin**, **Jetpack Compose**, **Coroutines**, **Flow**
- 🧱 **MVVM (Model-View-ViewModel)**  
- 💾 **Room Database**  
- 🌐 **Retrofit** для сетевых запросов  
- ⚙️ **Koin** для Dependency Injection
